# Nigeria-diaspora-passport-application
A user centred research and redesign for the Nigerian passport application and renewal experiences from abroad

**Status:** Active: discovery phase  
**Team:** Independent researchers  
**Started:** May 2026

## What this is

We are investigating how Nigerian citizens in the US, UK, and Canada 
experience the passport application and renewal process. We work in the 
open, sharing findings at each milestone.

## Read the project brief

[View the project brief](https://amogendukwe-png.github.io/diaspora-passport-study/brief/)

## Contents

- `brief/` — project brief and study overview
- `research/` — desk research, journey maps, interview guides and notes
- `prototype/` — AI-assisted service prototype (Phase 4)
- `outputs/` — published milestone outputs
